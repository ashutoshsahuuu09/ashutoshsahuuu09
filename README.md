<details>
<summary><b>🔷 Earthius — Intelligent RAG-Based Chatbot</b></summary>

<br/>

An end-to-end production-grade Retrieval-Augmented Generation chatbot engineered for semantic document intelligence. Earthius integrates Claude API with LangChain orchestration, FAISS vector indexing, FastAPI async backend, and MongoDB persistence — delivering sub-100ms API latency with 94% retrieval accuracy.

| Attribute | Details |
|-----------|---------|
| **Stack** | Claude API · LangChain · FAISS · FastAPI · MongoDB · Python |
| **Scale** | Multi-turn conversations · Optimized context windows |
| **Performance** | Sub-100ms API latency · 35% reduction in token usage |
| **Security** | Structured data validation · Error-handled ingestion pipeline |
| **Impact** | 94% semantic retrieval accuracy · Production-ready RAG architecture |
| **Repository** | [![GitHub](https://img.shields.io/badge/View_Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/ashutoshsahuuu09/Earthius) |

The system implements an optimized RAG pipeline with multi-turn conversation memory, intelligent context management, and scalable document ingestion. The architecture decouples retrieval from generation, enabling independent optimization of both layers. Token budgeting strategies and chunk-level caching were applied to achieve a 35% reduction in API token consumption without quality degradation.

</details>

<details>
<summary><b>🔷 Grid07 — Intelligent Query Routing System</b></summary>

<br/>

A semantic query routing infrastructure built on LangGraph that routes incoming queries to the optimal LLM handler using embedding-based similarity search. Grid07 reduces API costs by intelligently batching, caching, and routing requests — cutting misrouted queries by 87% and API costs by 42%.

| Attribute | Details |
|-----------|---------|
| **Stack** | LangGraph · Embedding Models · Python · FastAPI |
| **Scale** | High-throughput API gateway with intelligent routing logic |
| **Performance** | 87% reduction in misrouted queries · 42% API cost reduction |
| **Security** | Request validation · Graceful fallback routing |
| **Impact** | Production-ready semantic router · Significant LLM cost optimization |
| **Repository** | [![GitHub](https://img.shields.io/badge/View_Repo-181717?style=flat-square&logo=github&logoColor=white)](https://github.com/ashutoshsahuuu09/Grid07) |

Grid07 uses a graph-based orchestration model where each node represents a specialized LLM handler. Incoming queries are embedded and compared against handler signature embeddings via cosine similarity. The routing layer supports fallback chains, request batching by handler affinity, and a caching layer for high-frequency query patterns — drastically reducing redundant API calls.

</details>
